## 按任务找文件

| 你要改什么 | 先看 |
| --- | --- |
| URL 解析和任务创建 | `app/services/feature_service.py`、对应 `features/*_pack/pack.py` |
| 任务队列、暂停、恢复 | `app/services/core_service.py`、`app/bases/models.py` |
| HTTP 分块/续传/提速 | `features/http_pack/task.py` |
| 浏览器扩展连接桌面端 | `app/services/browser_service.py`、`browser_extension/app/src/background/desktop-bridge.ts` |
| 扩展资源捕获 | `browser_extension/app/src/background/resource-bridge.ts` |
| 配置项和设置页 | `app/supports/config.py`、`app/view/pages/setting_page.py` |
| 仓库工作流和构建验证 | `deploy.py`、`.github/workflows/build.yml`、`.github/workflows/crowdin.yml` |
