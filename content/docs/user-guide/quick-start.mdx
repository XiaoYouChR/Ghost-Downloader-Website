## 首次下载

1. 安装并打开 Ghost Downloader。
2. 复制一个公开 HTTP 文件链接。
3. 新建任务，选择保存目录。
4. 确认文件名、大小和保存位置。
5. 开始下载，并观察任务卡片状态。

## 扩展配对

1. 在桌面端设置页开启浏览器扩展服务。
2. 复制配对令牌。
3. 在扩展设置页粘贴令牌并保存。
4. 状态显示已连接后，即可把网页资源发送到桌面端。

## 平台要求

| 平台 | 要求 |
| --- | --- |
| Windows | `7 SP1+`，`x86_64` / `arm64` |
| macOS | `13.0+`，`x86_64` / `arm64` |
| Linux | `glibc 2.35+`，`x86_64` / `arm64` |

<Callout type="warn" title="Windows 7">
如果需要 Windows 7，请优先查看 Release 中标注的兼容版本。
</Callout>
