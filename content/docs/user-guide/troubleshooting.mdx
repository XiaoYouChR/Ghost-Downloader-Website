---
title: 常见问题
description: 用户侧最常见问题的快速排查表。
---

| 问题 | 先试 |
| --- | --- |
| 软件打不开 | 检查系统版本、杀毒拦截、安装包完整性和日志。 |
| 普通链接失败 | 换公开 HTTP 链接，排除代理、SSL、目录权限问题。 |
| 速度慢 | 调整预分块、自动提速、代理，或确认源站是否限速。 |
| BT 无速度 | 检查 DHT、tracker、P2P 网络环境和做种情况。 |
| M3U8 失败 | 确认清单未过期、headers 完整、运行时和 FFmpeg 可用。 |
| Bilibili 高画质不可用 | 检查 Cookie、清晰度权限和 FFmpeg。 |
| 扩展连不上 | 确认桌面端服务开启、令牌正确、端口 `14370` 未被占用。 |
| 文件打不开 | 确认任务完成、文件未移动、目标目录仍存在。 |
