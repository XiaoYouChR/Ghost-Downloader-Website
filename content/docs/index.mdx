Ghost Downloader 开发者文档面向四类任务：提交高质量 Issue、准备 Pull Request、开发 Feature Pack、快速理解仓库结构。

<Callout type="info" title="从任务开始">
反馈问题先看贡献入口和调试日志；准备改代码先看环境与运行、仓库地图和架构总览。
</Callout>
